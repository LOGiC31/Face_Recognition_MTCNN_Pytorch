The model uses dataset here named as 
'photos'.
You can add person with a specific person with folder name as their identity.
The model with process live video feed from your webcam, process it, detect faces using MTCNN, train the model using resnet by converting every image into its embedding and finally recognise them using facenet and triplet loss functions as evaluation parameters inbuilt into the model.
